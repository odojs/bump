
# bump
Easily bump the version of all the different package.json equivalents.

## Installation
    $ npm install -g tcoats/bump

## Usage
    $ bump patch
    
      patch'd
      0.6.9 -> 0.6.10
    
      files updated
      package.json
      bower.json
    
    $ bump minor
    
      minor'd
      0.6.9 -> 0.7.0
    
      files updated
      package.json
      bower.json
    
    $ bump major
    
      major'd
      0.6.9 -> 1.0.0
    
      files updated
      package.json
      bower.json
